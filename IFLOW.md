# Payloads All The Things - IFLOW.md

## 项目概述

Payloads All The Things 是一个包含大量有用有效载荷和绕过技术的Web应用程序安全资源库。该项目旨在为安全研究人员、渗透测试人员和漏洞赏金猎人提供各种漏洞类型的利用方法和测试有效载荷。

项目的主要特点：
- 包含各种Web漏洞类型的有效载荷和绕过技术
- 维护大量漏洞利用示例和方法
- 遵循特定的目录结构，每种漏洞类型都有独立的目录
- 为Burp Suite Intruder提供有效载荷文件

## 目录结构

项目遵循特定的目录结构，每种漏洞类型都有自己的目录，包含以下文件：
- `README.md`：漏洞描述和利用方法，包括各种有效载荷
- `Intruder`：提供给Burp Intruder的一组文件
- `Images`：README.md的图片
- `Files`：README.md中引用的文件

## 项目内容

该项目目前包含以下漏洞类型：

- 账户接管
- API密钥泄露
- 暴力破解速率限制
- 业务逻辑错误
- 点击劫持
- 命令注入
- CORS配置错误
- 跨站请求伪造
- SQL注入
- XSS注入
- XXE注入
- 以及其他数十种漏洞类型

## CVE漏洞利用

该项目还包括多个已知CVE漏洞的利用脚本，例如：
- Apache Struts 2漏洞利用
- Citrix漏洞利用
- Jenkins漏洞利用
- WebLogic漏洞利用
- 等等

## 贡献指南

该项目欢迎社区贡献，并遵循以下指南：
- 有效载荷必须经过清理
- 使用`id`和`whoami`进行RCE概念验证
- 使用`[REDACTED]`替换需要回调的域名
- 使用`10.10.10.10`和`10.10.10.11`作为IP地址示例
- 使用标准密码如`P@ssw0rd`、`Password123`等。

## 技术栈

该项目主要使用Markdown文件组织内容，并使用MkDocs构建网站。配置文件mkdocs.yml表明项目使用MkDocs的Material主题来部署文档网站。

## 用途

该项目主要用于：
- 渗透测试人员的参考手册
- 漏洞赏金猎人的工具集
- 安全研究人员的资源库
- 安全工具开发的参考
